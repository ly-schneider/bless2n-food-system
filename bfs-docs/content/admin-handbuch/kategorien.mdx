# Kategorien

Kategorien gruppieren Produkte im Shop (z. B. Getränke, Hauptgerichte, Desserts). Die Position bestimmt die Sortierreihenfolge.

---

# Kategorie erstellen

Im oberen Bereich der Seite:

- Name eingeben.
- Position festlegen (bestimmt die Reihenfolge in der Anzeige).

Nach dem Absenden erscheint die Kategorie in der Tabelle.

---

# Kategorie bearbeiten

Alle Felder sind direkt in der Tabelle editierbar:

- Name: Per Klick bearbeiten. Änderung wird bei Fokusverlust oder Enter gespeichert.
- Position: Zahl ändern, um die Sortierung anzupassen.

---

# Aktiv/Inaktiv

Per Toggle-Schalter kann eine Kategorie deaktiviert werden. Inaktive Kategorien und deren Produkte werden im Shop nicht angezeigt.

---

# Kategorie löschen

Über den Löschen-Button in der Tabelle. Eine Bestätigung wird angefordert.

<Callout type="warn">
Kategorien mit zugewiesenen Produkten können nicht gelöscht werden. Verschiebe die Produkte zuerst in eine andere Kategorie.
</Callout>
