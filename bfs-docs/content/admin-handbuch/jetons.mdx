# Jetons

Jetons sind farbige physische Tokens, die im POS-Jeton-Modus eingesetzt werden. Jeder Jeton hat einen Namen und eine Farbe. Beim Verkauf am POS wird angezeigt, welche Jetons dem Kunden übergeben werden müssen. Die Station erkennt anhand der Jeton-Farbe, welches Gericht ausgegeben wird.

---

# Jeton erstellen

<Steps>
### Name eingeben

Einen aussagekräftigen Namen wählen (z. B. "Hauptgericht", "Getränk").

### Farbe wählen

Aus 7 vordefinierten Farben wählen oder einen eigenen HEX-Wert eingeben:

| Farbe | HEX |
|---|---|
| Gelb | `#EAB308` |
| Blau | `#3B82F6` |
| Rot | `#EF4444` |
| Grün | `#22C55E` |
| Lila | `#A855F7` |
| Orange | `#F97316` |
| Grau | `#6B7280` |

### Jeton speichern

Der Jeton erscheint in der Liste und kann Produkten zugewiesen werden.
</Steps>

---

# Jetons Produkten zuweisen

Auf der Jetons-Seite befindet sich eine Zuweisungstabelle. Dort können einfache Produkte einem Jeton zugeordnet werden. Jedes Produkt kann genau einem Jeton zugewiesen werden.

---

# Jeton bearbeiten

Name und Farbe können jederzeit angepasst werden.

---

# Jeton löschen

Ein Jeton kann nur gelöscht werden, wenn er keinem Produkt zugewiesen ist. Die Anzahl der Zuweisungen wird neben dem Jeton angezeigt.

---

# POS-Ablauf mit Jetons

1. Kunde bestellt am POS.
2. Die POS-Oberfläche zeigt an, welche Jetons der Kunde erhält (Farbe und Anzahl).
3. Das Personal übergibt die entsprechenden Jetons.
4. An der Ausgabestation wird anhand der Farbe das richtige Gericht ausgegeben.

Siehe auch: [POS-Einstellungen](/admin-handbuch/pos) für die Moduswahl.
