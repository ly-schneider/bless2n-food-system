# Menüverwaltung

Ein Menü ist ein Bundle aus mehreren Slots. Jeder Slot bietet eine Auswahl an einfachen Produkten (z. B. Vorspeise, Hauptgang, Dessert).

---

# Menü erstellen

<Steps>
### Grunddaten eingeben

- Name: Menüname (max. 20 Zeichen).
- Preis: Gesamtpreis in CHF.
- Kategorie: Zuordnung zu einer [Kategorie](/admin-handbuch/kategorien).

### Bild hochladen (optional)

Wie bei Produkten: Drag & Drop oder Klick, max. 5 MB.

### Menü speichern

Das Menü wird erstellt und erscheint als Karte in der Übersicht.
</Steps>

---

# Slots verwalten

Slots definieren die Auswahlgruppen innerhalb eines Menüs (z. B. "Getränk", "Beilage").

- Slot hinzufügen: Neuen Slot mit einem Namen erstellen.
- Slot umbenennen: Name direkt in der Slot-Ansicht ändern.
- Reihenfolge ändern: Slots mit den Auf/Ab-Pfeilen verschieben.
- Slot löschen: Entfernt den Slot inklusive aller Optionen.

---

# Produktoptionen zuweisen

Innerhalb eines Slots können einfache Produkte als Optionen hinzugefügt werden. Bei der Bestellung wählen Kunden pro Slot eine der verfügbaren Optionen.

- Option hinzufügen: Aus der Liste der verfügbaren einfachen Produkte wählen.
- Option entfernen: Über den Entfernen-Button neben der Option.

---

# Menü bearbeiten

Auf der Menükarte können Grunddaten (Name, Preis, Kategorie, Bild) und Slots bearbeitet werden. Per Toggle kann ein Menü aktiviert/deaktiviert werden.

---

# Menü löschen

Über den Löschen-Button auf der Karte. Eine Bestätigung wird angefordert.
