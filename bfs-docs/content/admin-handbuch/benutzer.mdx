# Benutzerverwaltung

Unter Benutzer werden alle registrierten Konten aufgelistet.

---

# Benutzerliste

Die Tabelle zeigt:

- E-Mail: Anmeldeadresse des Benutzers.
- Name: Anzeigename (falls gesetzt).
- Rolle: Aktuelle Rolle (Admin, Customer).
- Erstellt am / Aktualisiert am: Zeitstempel.

Ein Klick auf einen Benutzer öffnet die Detailansicht.

---

# Benutzerdetails

Die Detailseite zeigt alle Informationen zum Konto: ID, E-Mail, Name, Rolle, Verifizierungsstatus und Zeitstempel.

---

# Rolle ändern

Über die Detailseite kann die Rolle eines Benutzers geändert werden. Verfügbare Rollen:

- Customer: Standard-Rolle für Bestellungen.
- Admin: Vollzugriff auf das Admin-Dashboard.

<Callout type="warn">
Rollenänderungen wirken sofort. Ein Entzug der Admin-Rolle sperrt den Benutzer umgehend aus dem Dashboard aus.
</Callout>

---

# Benutzer löschen

Im unteren Bereich der Detailseite befindet sich die Option zum Löschen. Die Löschung ist permanent und muss bestätigt werden.

---

# Neue Admins hinzufügen

Um einen neuen Admin hinzuzufügen, wird eine Einladung per E-Mail versendet. Siehe [Einladungen](/admin-handbuch/einladungen).
