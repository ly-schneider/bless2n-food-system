# Dashboard

Das Dashboard zeigt eine Übersicht der Bestell- und Umsatzdaten im gewählten Zeitraum.

---

# Datumsbereich wählen

Oben rechts befindet sich ein Datumsbereich-Selektor. Standardmässig werden die letzten 30 Tage angezeigt. Start- und Enddatum können über den Kalender angepasst werden.

---

# Bestellungen pro Tag

Ein Balkendiagramm zeigt die Anzahl der Bestellungen pro Tag im gewählten Zeitraum. Jeder Balken entspricht einem Tag.

---

# Umsatz pro Tag

Ein Liniendiagramm zeigt den Tagesumsatz in CHF. Die Werte werden als Linie dargestellt, um Trends sichtbar zu machen.

---

# Tiefe Bestände

Unterhalb der Diagramme zeigt eine Liste die Top 5 Produkte mit tiefem Bestand. So lässt sich auf einen Blick erkennen, welche Produkte bald nachgefüllt werden müssen.

<Callout type="warn">
Produkte mit einem Bestand von 0 werden im Shop nicht mehr als verfügbar angezeigt. Bestände können unter [Produkte](/admin-handbuch/produkte) angepasst werden.
</Callout>
